#### 3.1.2 Установка GNS3 VM для VirtualBox

Для расширения функциональности GNS3 и возможности эмуляции более сложных сетевых устройств была установлена GNS3 VM.

**Процесс установки:**

1. **Скачивание образа** - архив с образом виртуальной машины GNS3 VM был скачан с официального сайта GNS3

2. **Распаковка архива** - извлечение файла `GNS3 VM.ova` из архива

3. **Импорт в VirtualBox:**
   - Запуск Oracle VM VirtualBox
   - Выбор меню: **Файл → Импорт конфигураций**
   - Указание пути к распакованному образу `GNS3 VM.ova`

4. **Настройка параметров импорта:**
   - В политике MAC-адресов выбрано: **«Сгенерировать новые MAC-адреса всех сетевых адаптеров»**
   - Ожидание завершения процесса импорта

**Рис. 3.3. Настройка параметров импорта конфигураций ВМ**


5. **Настройка виртуальной машины:**
   - В VirtualBox выбрана импортированная виртуальная машина
   - Переход в меню: **Машина → Настроить**
   - Исправление ошибок согласно рекомендациям системы

6. **Проверка минимальных ресурсов:**
   - Оперативная память: **2048 МБ** (минимальное требование)
   - Процессоры: **2 ядра**

7. **Настройка вложенной виртуализации (Nested Virtualization):**
   - В графическом интерфейсе опция **Включить Nested VT-x/AMD-V** отсутствовала
   - Для включения использована командная строка:

```bash
vboxmanage modifyvm "GNS3 VM" --nested-hw-virt on
